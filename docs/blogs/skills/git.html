# 常用git命令

# git remote

  • git remote：此命令列出本地仓库关联的所有远程仓库的名称。
  • git remote -v：此选项显示所有远程仓库的 URL，以及它们对应的名称。
  • git remote add ：该子命令向本地仓库添加一个新的远程仓库。 是您为远程仓库选择的自定义名称， 是远程仓库的 URL。
  • git remote rename ：该子命令重命名现有的远程仓库。 是当前的远程仓库名称， 是您要为其指定的新名称。
  • git remote remove ：该子命令取消本地仓库与远程仓库之间的关联。 是要删除的远程仓库的名称。
  • git remote set-url ：该子命令更改现有远程仓库的 URL。 是远程仓库的名称， 是要指定的新 URL。
